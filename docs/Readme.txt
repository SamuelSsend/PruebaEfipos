INSTRUCCIONES PARA PUESTA EN MARCHA DE LA PLATAFORMA.
1. Subir código fuente al servidor.
2. Importar base de datos sql.
3. Modificar el archivo ".env" para la conexión de la base de datos, en el apartado "DB_CONNECTION"
4. Para arrancar el proyecto en la terminal debes introducir el comando "php artisan serve".
